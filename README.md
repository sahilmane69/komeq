# Komeq – IoT Self Balancing Robot

Komeq is a **UI/UX display website for an IoT Self-Balancing Robot project**.  
The website is designed to present the robot concept, design, and features in a clean and modern way.

🌐 Live Demo: https://komeq.vercel.app

## About the Project

This project focuses on creating a **simple presentation website** for an IoT self-balancing robot.  
It visually represents the idea, design, and working concept of the robot for demonstration and project showcase.

## Tech Used

- Next.js
- React
- Modern UI / UX design
- Vercel (Deployment)

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
